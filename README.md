## How to run
- **All-in-one with Docker:** `docker-compose up --build` then open http://localhost:3000
- **Local dev:** start the backend, then `npm run dev` in the frontend and open the Vite URL.

DocStream is built to feel premium from the first click: fast, clean, and secure.
